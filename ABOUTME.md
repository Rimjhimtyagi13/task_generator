## About This Project

**Tasks Generator (Mini Planning Tool)** is a web application that helps transform a high-level feature idea into:
- structured user stories,
- engineering tasks,
- and a lightweight planning spec.

The app allows users to:
- input a feature goal, target users, and constraints,
- generate tasks using an LLM (or mock mode),
- edit the generated tasks,
- view the last 5 generated specs,
- and export results for further use.

The project prioritizes:
- simplicity,
- clear UX,
- clean code structure,
- and safe AI usage.

---

## AI Usage Philosophy

AI is used as a **helper**, not a replacement for reasoning.  
All prompts, outputs, and flows were reviewed manually, and mock outputs are supported to ensure the app works even without an API key.
